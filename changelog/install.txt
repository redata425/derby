Derby 4.2.3
--------------------

This version has updated all auto-completions to be in line with Bourbon 4.2.3 and Neat 1.7.2.

	- Mixins that are deprecated or will soon be deprecated have been removed.
    - Removed docs feature
    - Versioning will now stay the same as Bourbon's version numbers
    - Now support for both SCSS and SASS

Report issues at https://github.com/rossedman/derby

Derby 1.0.0
--------------------

This version added very few features as Bourbon and Neat have not changed much in the past
few months. 

	- Neat variables were added 
	- Bourbon / Neat typos were fixed.

Report issues at https://github.com/rossedman/derby
